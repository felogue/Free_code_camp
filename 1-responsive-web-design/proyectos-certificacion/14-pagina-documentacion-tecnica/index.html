What is HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear páginas web. Es un lenguaje de marcado que permite definir la estructura y el contenido de una página web, así como su presentación visual. En esta sección, presentaremos una breve introducción a HTML y sus características básicas.

HTML fue creado en 1990 por Tim Berners-Lee, el padre de la World Wide Web. Desde entonces, ha evolucionado a través de varias versiones, con la última versión siendo HTML5.

HTML no es un lenguaje de programación, sino un lenguaje de marcado que se utiliza para estructurar el contenido de una página web. Los elementos HTML se utilizan para definir diferentes partes de una página web, como encabezados, párrafos, listas, imágenes y enlaces. Los navegadores web interpretan el código HTML y lo muestran en forma de página web.

Basic HTML Structure
  • Todo documento HTML debe comenzar con la etiqueta <!DOCTYPE html> seguida de la etiqueta <html>.
  • El contenido de la página web se encuentra entre las etiquetas <head> y <body>.
  • La etiqueta <head> contiene información sobre la página web, como el título y los metadatos.
  • La etiqueta <body> contiene el contenido visible de la página web.
HTML Headings

El uso de encabezados y párrafos, son dos elementos esenciales para organizar y dar formato al texto en una página web.

Los encabezados, también conocidos como headings, se utilizan para crear títulos y subtítulos en una página web. Permiten organizar la información de forma jerárquica y facilitar la lectura del contenido.

Existen seis niveles de encabezados en HTML, del 1 al 6. Cada nivel se representa con una etiqueta diferente. El 1 es el encabezado más grande y el 6 el más pequeño.

Ejemplo de código:

<h1>Título principal</h1>
<h2>Subtítulo 1</h2>
<h3>Subtítulo 2</h3>
<h4>Subtítulo 3</h4>
<h5>Subtítulo 4</h5>
<h6>Subtítulo 5</h6>

Consejos para usar encabezados:

  • Utiliza un encabezado de nivel superior para el título principal de la página.
  • Usa encabezados de niveles inferiores para los subtítulos y las secciones de la página.
  • Sigue una lógica jerárquica al usar los encabezados.
  • No uses demasiados encabezados del mismo nivel.
HTML Paragraphs

Los párrafos son bloques de texto que se utilizan para presentar la información principal en una página web.

Ejemplo de código

<p>Este es un párrafo de texto.</p>

Consejos para usar párrafos:

  • Comienza cada párrafo con una oración inicial que resuma el tema del párrafo.
  • Mantén los párrafos concisos y fáciles de leer.
  • Utiliza sangrías para separar los párrafos.
  • No uses demasiados saltos de línea dentro de un párrafo.
Images in HTML

HTML proporciona varias formas de trabajar con imágenes en una página web. Las imágenes pueden mejorar la apariencia y la experiencia del usuario en un sitio web, y es importante entender cómo insertar y ajustar imágenes correctamente en HTML.

La forma más común de insertar una imagen en HTML es utilizando el elemento <img>. Este elemento no requiere una etiqueta de cierre y tiene varios atributos clave:

  • src: especifica la URL o la ruta de la imagen
  • alt: proporciona un texto alternativo para la imagen, que se muestra si la imagen no se carga correctamente
  • width y height: establecen el ancho y la altura de la imagen en píxeles

A continuación, se muestra un ejemplo de cómo utilizar el elemento <img>:

<img src="imagen.jpg" alt="Descripción de la imagen" width="300" height="200">

Formatos de Imagen

HTML admite varios formatos de imagen, incluyendo JPEG, PNG y GIF. La elección del formato depende del tipo de imagen y de los requisitos de calidad y compresión. Algunos puntos clave sobre los formatos de imagen son:

  • JPEG (.jpg) es adecuado para fotografías y ofrece una buena calidad de imagen con una compresión efectiva
  • PNG (.png) es ideal para imágenes con áreas transparentes y preserva la calidad sin pérdidas, pero puede generar archivos más grandes
  • GIF (.gif) es adecuado para imágenes animadas y utiliza una paleta de colores limitada

Atributo CSS background-image

Además del elemento <img>, también puedes utilizar el atributo CSS background-image para mostrar una imagen de fondo en un elemento HTML. Esto te permite tener más control sobre la posición y el tamaño de la imagen de fondo. A continuación, se muestra un ejemplo:

<div style="background-image: url(imagen.jpg); width: 500px; height: 300px;"></div>

Conclusiones

Las imágenes desempeñan un papel importante en el diseño y la experiencia del usuario en una página web. Con HTML, puedes insertar imágenes utilizando el elemento <img> y personalizar su apariencia con CSS. Recuerda optimizar tus imágenes para un rendimiento web óptimo y proporcionar siempre un texto alternativo adecuado para mejorar la accesibilidad.

HTML Forms

Introducción

Los formularios son una parte fundamental de la web, ya que permiten a los usuarios interactuar con los sitios web enviando datos al servidor. En HTML, los formularios se crean utilizando elementos como <form>, <input>, <textarea>, entre otros.

Estructura básica de un formulario

Un formulario en HTML se estructura dentro de un elemento <form>, que puede contener elementos de entrada de datos como campos de texto, casillas de verificación, botones de opción y botones de envío. Cada campo de entrada se define con el elemento <input> o <textarea>.

Atributos comunes de los elementos de formulario

  • name: Nombre del campo que se enviará al servidor.
  • id: Identificador único del campo.
  • type: Tipo de campo de entrada (texto, email, contraseña, etc.).
  • required: Indica que un campo es obligatorio.
  • placeholder: Texto de ejemplo que se muestra en el campo antes de que se introduzca información.
  • value: Valor predeterminado del campo.

Enviando datos al servidor

Cuando un usuario completa un formulario y hace clic en el botón de envío (<input type="submit">), los datos se envían al servidor especificado en el atributo action del elemento <form>. La solicitud HTTP resultante puede ser manejada por un servidor web para procesar la información recibida.

Conclusiones

Los formularios en HTML son una herramienta poderosa para interactuar con los usuarios y recopilar información. Es importante diseñar formularios accesibles y fáciles de usar para mejorar la experiencia del usuario en la web.

HTML Tables

Introducción

Las tablas en HTML permiten organizar datos de manera estructurada en filas y columnas. Son útiles para mostrar información de forma ordenada y facilitar su lectura.

Elementos de una tabla

Una tabla en HTML se crea utilizando los siguientes elementos:

  • <table>: Elemento contenedor de la tabla.
  • <tr>: Define una fila de la tabla.
  • <th>: Define una celda de encabezado.
  • <td>: Define una celda de datos.

Ejemplo de una tabla simple en HTML

Nombre Edad
María 30
Juan 25

Estilos de tabla

Puedes aplicar estilos CSS a las tablas para personalizar su apariencia, como colores de fondo, bordes y espaciado entre celdas.

Conclusiones

Las tablas en HTML son una herramienta útil para mostrar datos de forma organizada y estructurada. Es importante utilizarlas de manera adecuada para mejorar la presentación de la información en una página web.
